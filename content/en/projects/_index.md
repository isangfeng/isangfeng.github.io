## sfRehearsal

A few days ago, I was preparing an English presentation. During rehearsal, I found that it would be much useful if I had a reference voice to follow and could record my own speech for comparison. This inspired me to create **sfRehearsal**, a simple tool for presentation practice and pronunciation training.

For the repository URL and more details, please see: [https://github.com/isangfeng/sfRehearsal.git](https://github.com/isangfeng/sfRehearsal.git)
